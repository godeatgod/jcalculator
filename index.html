jCalculator

Awesome jQuery plugin for calculus inputs

Download

How to install

1. Include the following files before body tag close.

<link href="path_to/jCalculator.css" rel="stylesheet" type="text/css">
<-- jQuery library -->
<script type="text/javascript" src="path_to/jQuery.js"></script>
<script type="text/javascript" src="path_to/jCalculator.js"></script>

2. Call the jCalculator plugin.

<script type="text/javascript">
$('#yourInput').jCalculator({ <-- #yourInput its the input id -->
position: top <-- position can be top or bottom -->
});
</script>
